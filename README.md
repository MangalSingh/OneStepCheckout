# OneStepCheckout
Magento 2 One Step Checkout Extension by Octocub

Octocub Magento 2 One Step Checkout is a performance-focused checkout extension that simplifies the default Magento checkout into a single, user-friendly page. By reducing steps and distractions, it helps customers complete purchases faster, improves user experience, and can increase conversion rates—especially on mobile devices.

This extension is built for modern Magento stores and supports flexible customization so merchants can match the checkout layout to their brand while keeping the flow clean and intuitive.

Installation

Run the following commands from your Magento root directory:

bin/magento module:enable Octocub_OneStepCheckout
bin/magento setup:upgrade
bin/magento cache:flush
bin/magento setup:di:compile
bin/magento setup:static-content:deploy -f

Configuration

Go to:

Stores → Configuration → Sales → Octocub One Step Checkout

From here you can enable the module and configure checkout layout, delivery options, and other settings.
